1)¿Qué versiones de JavaScript hay en el mercado?

Las versiones que hay en el mercado son: JavaScript 1.0, 1.1, 1.2, 1.3, 1.4, 1.5, 1.6, 1.7. Todos los navegadores modernos soportan ECMAScript 5.1, los navegaodres mas viejos soportan ECMAScript 3, tambien estan las versiones ECMAScript 1, 2, 3, 5, 6 y 7.

2) ¿Cómo se integra JavaScript a HTML?

Hay 3 formas de integrar js a html, las cuales son: con la etiqueta script. <head><script>console.log (mostando);</script></head>,
<head><script src="JS/script.js"> </script></head> y <body><script>console.log(mostrando);</script></body>.

3) ¿Cómo es el manejo de las variables, tipos de datos y expresiones regulares en JS?

Las variables se pueden declarar usando var, const, o let. var tiene alcance de función, mientras que const y let tienen alcance de bloque. Las variables const no pueden ser reasignadas, mientras que las variables let si pueden serlo. Seis tipos de datos primitivos, controlados por el operador typeof los cuales son: "undefined", "boolean", "number", "string", "bigint", "symbol", tambien estan los datos: null, objet, function. Existen dos formas de definir una expresión regular: una de ellas es por una cadena literal y mediante el constructor de objeto RegExp. Una expresión regular se puede aplicar a diferentes cadenas mediante expReg, test(cadena), que devuelve true (verdad) si cadena cumple expReg y false (falso) en caso contrario.

5) ¿Cuáles son los operadores y conversión entre tipos?

Los operadores existen son tres tipos: Aritméticos, Relacionales y Lógicos. Los Aritméticos son: suma(+), resta(-), multiplicacion(*), division(/) y modulo (%). Los relacionales son: mayor que ( > ), menor que ( < ), mayor o igual ( >= ), menor o igual ( <= ), igual que ( == ) y distinto de ( != ). Los logicos son: AND(&&) y OR(||). Hay 3 conversiones las cuales son: ToString: ocurre cuando se muestra algo, se puede realizar con String(value). ToNumber: Ocurre en operaciones matemáticas, se puede realizar con Number(value). ToBoolean: Ocurren en operaciones lógicas, se puede realizar con Boolean(value).

6) ¿Cuáles son las estructura de control de JS?

Una estructura de control es una herramienta que tienen los lenguajes de programación y que se utiliza para controlar el flujo de las operaciones durante el programa. Las estructuas son: IF/ELSE, SWITCH, FOR, WHILE y DO/WHILE.
