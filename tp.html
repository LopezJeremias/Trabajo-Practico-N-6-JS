3) ¿Cómo es el manejo de las variables, tipos de datos y expresiones regulares en JS?

Las variables se pueden declarar usando var, const, o let. var tiene alcance de función, mientras que const y let tienen alcance de bloque. Las variables const no pueden ser reasignadas, mientras que las variables let si pueden serlo. Seis tipos de datos primitivos, controlados por el operador typeof los cuales son: "undefined", "boolean", "number", "string", "bigint", "symbol", tambien estan los datos: null, objet, function. Existen dos formas de definir una expresión regular: una de ellas es por una cadena literal y mediante el constructor de objeto RegExp. Una expresión regular se puede aplicar a diferentes cadenas mediante expReg, test(cadena), que devuelve true (verdad) si cadena cumple expReg y false (falso) en caso contrario.

5) ¿Cuáles son los operadores y conversión entre tipos?

Los operadores existen son tres tipos: Aritméticos, Relacionales y Lógicos. Los Aritméticos son: suma(+), resta(-), multiplicacion(*), division(/) y modulo (%). Los relacionales son: mayor que ( > ), menor que ( < ), mayor o igual ( >= ), menor o igual ( <= ), igual que ( == ) y distinto de ( != ). Los logicos son: AND(&&) y OR(||). Hay 3 conversiones las cuales son: ToString: ocurre cuando se muestra algo, se puede realizar con String(value). ToNumber: Ocurre en operaciones matemáticas, se puede realizar con Number(value). ToBoolean: Ocurren en operaciones lógicas, se puede realizar con Boolean(value).

6) ¿Cuáles son las estructura de control de JS?

Una estructura de control es una herramienta que tienen los lenguajes de programación y que se utiliza para controlar el flujo de las operaciones durante el programa. Las estructuas son: IF/ELSE, SWITCH, FOR, WHILE y DO/WHILE.

7) ¿Qué es el objeto dom en JS? Y ¿Cómo trabaja?

El (DOM) es un modelo de objeto de documento, es una interfaz de programación para los documentos HTML y XML. Este modelo facilita una estructurada representación del documento y define la manera en que los programas pueden acceder, al fin de modificar, tanto su estructura, estilo y contenido.

8) ¿Qué función tiene en JS el Object Map?

La funcion que cumple es que te permite iterar sobre un arreglo y modificar sus elementos utilizando una función callback. La función callback lo que hara es que se ejecutará entonces en cada uno de los elementos del arreglo.

9) ¿Cuáles son todos los eventos que trabaja JS?

Los eventos que trabaja js son: onblur: Un elemento pierde el foco, <button>, <input>, <label>, <select>, <textarea>, <body>. onchange: Un elemento ha sido modificado, <input>, <select>, <textarea>. onclick: Pulsar y soltar el ratón, Todos los elementos ondblclick: Pulsar dos veces seguidas con el ratón, Todos los elementos. onfocus: Un elemento obtiene el foco, <button>, <input>, <label>, <select>, <textarea>, <body>. onkeydown: Pulsar una tecla y no soltarla, Elementos de formulario y <body>. onkeypress: Pulsar una tecla, Elementos de formulario y <body>. onkeyup: Soltar una tecla pulsada, Elementos de formulario y <body>. onload: Página cargada completamente, <body>. onmousedown: Pulsar un botón del ratón y no soltarlo, Todos los elementos. onmousemove: Mover el ratón, Todos los elementos. onmouseout: El ratón "sale" del elemento, Todos los elementos. onmouseover: El ratón "entra" en el elemento, Todos los elementos. onmouseup: Soltar el botón del ratón, Todos los elementos. onreset: Inicializar el formulario, <form>. onresize: Modificar el tamaño de la ventana, <body>. onselect: Seleccionar un texto, <input>, <textarea>. onsubmit: Enviar el formulario, <form>. onunload: si abandona la página, por ejemplo al cerrar el navegador <body>.
